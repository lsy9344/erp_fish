# ERP Fish 추가 기능 견적서

## 1. 견적 목적

2026-06-05 회의에서 정리된 추가 기능과 기존 엑셀 기반 운영 기능 보완 작업에 대한 간단 견적입니다.

기존 1차 개발 범위는 지점 일일 장부 입력, 본사 관제, 마감/정정, 기본 리포트까지였습니다. 아래 항목은 실제 운영에 맞추기 위해 추가 또는 고도화가 필요한 기능입니다.

## 2. 견적 내역

| 구분 | 작업 내용 | 금액 |
| --- | --- | ---: |
| 1 | 본사 전체 지점 통합 재고 화면 | 100,000원 |
| 2 | 상품별 분석, 최고매출품목, 리포트 고도화 | 200,000원 |
| 3 | 직원 관리, 월간 근무표, 급여 정산 기초 기능 | 200,000원 |
| 4 | 본사 고정비 입력 및 월 손익계산서 | 300,000원 |
| 5 | 지점장 화면 회계 지표 숨김, 손실액 계산 정책 반영, 작성자명 저장 등 보완 | 100,000원 |
| 6 | LINE 또는 텔레그램 자동 알림 | 200,000원 |
|  | **추가 견적 합계** | **1,100,000원** |

## 3. 무료 제공 항목

| 항목 | 금액 |
| --- | ---: |
| 이카운트 엑셀 업로드 기반 매입 자동 생성 | 0원 |

이카운트 엑셀 업로드 기능은 추가견적에 포함하지 않고 무료 제공 항목으로 처리합니다.

## 4. 포함 범위

- 전체 지점의 품목별 재고 합산 화면
- 직원별 근무일수와 급여 정산 참고 자료
- 월세, 세금 등 고정비 입력 후 월 손익 확인
- 본사용 상품/기간/월별 리포트 보강
- LINE 또는 텔레그램을 통한 주요 알림 발송

## 5. 제외 범위

- AI 분석 기능 구현
- POS, 카드사, 은행 자동 연동
- 과거 전체 엑셀 데이터 일괄 이관
- 서버 이용료, LINE 공식 계정 또는 텔레그램 운영 비용
- 오픈 후 장기 유지보수 비용

AI 분석은 이번 개발 범위에 포함하지 않습니다. 향후 데이터가 충분히 쌓인 뒤 별도 고도화 단계에서 논의하는 것을 권장합니다.

## 6. 일정

예상 개발 기간은 약 3주입니다.

세부 정책 확정, 테스트 피드백, 외부 알림 계정 준비 상황에 따라 일정은 조정될 수 있습니다.

## 7. 안내 사항

- 본 견적은 간단 견적 초안입니다.
- 부가세가 필요한 경우 별도입니다.
- LINE 알림은 기술적으로 가능하지만, LINE Notify가 종료되었기 때문에 LINE 공식 계정과 Messaging API를 사용해야 합니다.
- LINE 또는 텔레그램의 실제 운영 비용은 각 서비스의 정책에 따라 별도 확인이 필요합니다.
