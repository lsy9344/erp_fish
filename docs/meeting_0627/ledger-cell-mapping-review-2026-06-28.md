# 전체 셀 반영 검증표 (WO-17)

- 작성일: 2026-06-28
- 원본 장부: `docs/reference_from_customer/장부-202605현대 (1).xlsx`, 시트 `26`
- 목적: 고객 장부의 주요 셀이 앱 화면/계산에 어떻게 반영되는지 표로 확인한다. 이 문서는 **새 계산을 구현하는 작업이 아니라**, WO-14(본사 홈 이중 매출/마진) 착수 전에 누락과 차이를 드러내는 검증 산출물이다.
- 선행 관계: WO-14는 이 표에서 "본사 홈 반영 범위"가 정리된 뒤 착수한다.

## 0. 읽는 법

- `반영 상태` 값
  - `확인됨`: 셀 의미·수식이 원본에서 확인됐고 앱 계산 대응이 명확하다.
  - `라벨 확인 필요`: 계산값은 앱에 있으나 화면 라벨이 장부 용어와 다를 수 있다. WO-14에서 라벨을 맞춘다.
  - `앱 확인 필요`: 본사 아이디로 `홈`/리포트 화면을 직접 열어 현재 표시값과 위치를 확인해야 한다. (구현자가 실제 앱에서 채운다.)
  - `의도적 제외`: 앱에 반영하지 않기로 한 셀. 제외 사유를 함께 남긴다.
- 마진율 반전 표시(`100% - 표시값`)는 **본사 홈 display 전용**이다. 저장값·리포트·이상 신호 계산값은 덮어쓰지 않는다.

- `AE4`, `AE5`, `C5`, `C17`의 의미는 확인됐다. `AE4`를 다시 의미 미확정 항목으로 되돌리지 않는다.
- 앱의 `79.5%` 류 표시는 **표시 방향만 반대로** 하여 `20.5%`로 보여준다(`100% - 표시값`). 계산 저장값은 바꾸지 않는다.

## 2. WO-14 착수 전 채워야 할 셀 (구현자 작업분)

아래 행은 구현자가 본사 아이디로 `홈`/리포트 화면을 직접 열고, 원본 `26` 시트의 해당 셀과 대조해 `현재 값`/`기대 값`/`차이`/`조치`를 채운다. `AE4/AE5/C5/C17` 외에 고객이 홈/리포트에서 실제로 보는 주요 셀을 대상으로 한다.

| 기준 셀 | 의미(추정) | 수식(원본) | 원천 데이터 | 현재 화면 위치 | 현재 값 | 기대 값 | 차이 | 조치 |
| --- | --- | --- | --- | --- | --- | --- | --- |
| `C15` | 장부 매출이익 | _원본 확인_ | 장부 매출 - 장부 원가 | 본사 홈 | _앱 확인 필요_ | _원본 값_ | | |
| `AG36/AG63/AG76` | 분석 원가(품목군별) | _원본 확인_ | FIFO/추정 원가 | 리포트 | _앱 확인 필요_ | _원본 값_ | | |

## 4. 본사 홈 vs 월별/리포트 구분

- 본사 홈에 필요한 셀: `C5/C17`(장부 기준), `AE4/AE5`(분석 기준). WO-14에서 두 기준을 별도 필드로 나란히 표시하고, 마진율은 `100% - 표시값`으로 반전 표시한다.
- 월별/리포트에 필요한 셀: 품목군별 매출/원가(`AI*`, `AG*`), 결제수단별 매출(`C22~C24`, `C36`). WO-15 월별 손익/품목매출 시트와 연결한다.
